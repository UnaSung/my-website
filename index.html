Home Run Taiwan 台灣特色地圖設計 Home Run Taiwan Map Interface Design

專案概述 Project Overview

負責 Home Run Taiwan 平台的使用者介面設計，以台灣特色地圖為核心，創造富有在地特色的視覺體驗。運用插畫風格設計，為平台注入活潑且親和的設計語彙。 Responsible for the user interface design of Home Run Taiwan platform, focusing on Taiwan's feature map as the core element, creating a visually engaging experience with local characteristics. Utilizing illustration style to inject lively and friendly design language.

設計理念 Design Concept

以深綠色為主色調，營造自然清新的視覺感受。地圖採用淺綠色系，搭配水波紋理和海洋元素，突出台灣海島特色。周邊點綴荷葉、魚群等插畫元素，增添趣味性和在地特色。整體設計強調視覺層次感，讓使用者能直覺地理解地圖資訊。 Using deep green as the main color to create a natural and fresh visual experience. The map features a light green palette, complemented by wave patterns and ocean elements to highlight Taiwan's island characteristics. Decorative elements like lotus leaves and fish illustrations add playfulness and local flavor. The overall design emphasizes visual hierarchy for intuitive map information understanding.

設計重點 Design Focus

• 客製化地圖視覺設計 • 獨特插畫風格定義 • 視覺層次安排規劃 • 色彩計畫與應用 • 裝飾元素設計 • 介面視覺優化 • Customized map visual design • Unique illustration style definition • Visual hierarchy planning • Color scheme and application • Decorative elements design • Interface visual optimization

視覺設計細節 Visual Design Details

精心設計每個視覺元素，包含地圖輪廓的線條處理、區域分區的明確視覺區隔、水波紋的圖樣設計，以及周邊裝飾元素的配置。特別注重元素間的和諧與平衡，確保整體視覺的一致性。 Carefully designed each visual element, including map outline treatment, clear visual separation of regions, wave pattern design, and arrangement of decorative elements. Special attention was paid to harmony and balance between elements to ensure overall visual consistency.
